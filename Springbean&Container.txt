Q/- What is a Spring Bean and Spring container? How to activate the Spring Container in our
application.

A1/-
Spring Bean:
anj simple java class whoes associate with Spring container software is known as Spring bean.
and spring is POJO class.

Spring container:
So in spring application,a one class is controlling all the spring beans.
That class is known as spring container.
we dont need to install this spring container.
This is also very lightweight and activate in any java application

To activate this Spring Container

we have to add Spring Jar file in our class path using maven and then we have to create one
xml file and in that we have to create Spring bean.
